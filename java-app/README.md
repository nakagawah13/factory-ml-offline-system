# Factory ML Offline System

## 概要
このプロジェクトは、工場内のオフライン環境で稼働する構造化データを用いた機械学習システムです。エンジニアが管理用GUIアプリを操作し、推論、シミュレーション、モデル再学習、劣化診断を行うことができます。

## プロジェクト構成
- **java-app**: Javaで実装されたGUIアプリケーション
  - **src/main/java/com/factory/ml**: アプリケーションのソースコード
    - **controller**: 各タブのコントローラー
    - **service**: ビジネスロジックを提供するサービス
    - **model**: データモデルクラス
    - **util**: ユーティリティクラス
  - **src/main/resources**: FXMLファイルとスタイルシート
  - **pom.xml**: Mavenプロジェクトの設定ファイル
  - **README.md**: Javaアプリケーションのドキュメント

## 主な機能
- **推論タブ**: データの読み込み、バリデーション、推論実行
- **学習タブ**: トレーニングデータの選択、バリデーション、モデルのトレーニング
- **シミュレーション機能**: What-Ifシミュレーションのインターフェース
- **分析レポート表示**: 生成されたレポートを表示

## 技術スタック
- **Java**: アプリケーションのメイン言語
- **JavaFX**: GUIフレームワーク
- **Maven**: プロジェクト管理ツール

## セットアップ
1. Mavenを使用して依存関係を解決します。
2. `pom.xml`を使用してプロジェクトをビルドします。
3. `FactoryMLApp.java`を実行してアプリケーションを起動します。

## 使用方法
アプリケーションを起動した後、推論タブまたは学習タブを選択し、必要な操作を行います。データを読み込み、バリデーションを実行し、推論またはトレーニングを行うことができます。
