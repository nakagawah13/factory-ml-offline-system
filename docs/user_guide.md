# ユーザーガイド

## 概要
このユーザーガイドは、工場内のオフライン環境で稼働する構造化データを用いた機械学習システムの使用方法について説明します。このシステムは、エンジニアがデータの推論、シミュレーション、モデルの再学習、劣化診断を行うためのGUIアプリケーションを提供します。

## システム要件
- **ハードウェア要件**
  - Intel Core i7/i5 (12th Gen)
  - 16GB RAM
  - Windows OS

- **ソフトウェア要件**
  - Java 17+
  - Python 3.10+
  - 必要なライブラリは、`requirements.txt`および`pom.xml`に記載されています。

## インストール手順
1. **Javaアプリケーションのビルド**
   - Mavenを使用して、`java-app`ディレクトリ内で以下のコマンドを実行します。
     ```
     mvn clean install
     ```

2. **Pythonトレーナーのセットアップ**
   - `python-trainer`ディレクトリ内で以下のコマンドを実行して、必要なライブラリをインストールします。
     ```
     pip install -r requirements.txt
     ```

## アプリケーションの起動
- Javaアプリケーションを起動するには、以下のコマンドを実行します。
  ```
  java -jar target/FactoryMLApp.jar
  ```

## 使用方法

### 1. 推論タブ
- **データの読み込み**
  - `[Load Data]`ボタンをクリックして、CSVまたはExcelファイルを選択します。
  
- **バリデーション**
  - データが読み込まれると、自動的にバリデーションが実行されます。成功した場合は、データプレビューが表示され、`[Run Inference]`ボタンが活性化されます。

- **推論の実行**
  - `[Run Inference]`ボタンをクリックして、推論を実行します。結果は画面に表示されます。

### 2. 学習タブ
- **トレーニングデータの選択**
  - `[Select Training Data]`ボタンをクリックして、過去のデータを選択します。

- **モデルのトレーニング**
  - `[Start Training]`ボタンをクリックして、モデルのトレーニングを開始します。トレーニングが完了すると、生成されたレポートが表示されます。

### 3. シミュレーションタブ
- **What-Ifシミュレーション**
  - `[Simulation]`ボタンをクリックして、選択したデータ行に対するシミュレーションを開始します。スライダーを使用して値を変更し、リアルタイムで推論結果を確認できます。

## モデル管理
- モデルは`/models/archive/`ディレクトリに保存され、バージョン管理が行われます。新しいモデルを適用する際は、`app_settings.json`で現在のモデルパスを更新します。

## トラブルシューティング
- **エラーが発生した場合**
  - エラーメッセージを確認し、必要に応じてデータの形式やスキーマを見直してください。

## 追加機能
- システムには、データドリフト検知やSHAP分析を行うための分析レポート生成機能が含まれています。これにより、モデルのパフォーマンスを継続的に監視できます。
